# TokoSepatu

Aplikasi e-commerce toko sepatu sederhana berbasis PHP Native.

## Fitur

### Customer
- Registrasi dan login
- Lihat daftar produk
- Cari produk
- Filter berdasarkan kategori dan harga
- Detail produk
- Keranjang belanja
- Checkout
- Riwayat transaksi
- Upload foto profil

### Admin
- Login admin
- Dashboard dengan grafik transaksi
- Kelola produk (tambah, edit, hapus)
- Lihat detail transaksi
- Update status transaksi
- Preview toko

## Teknologi

- PHP
- MySQL
- Bootstrap 5
- Chart.js
- Font Awesome
